העברת קבצים מ-Google Drive ל-GCS

כלי מלא להעברת קבצים ותיקיות מ-Google Drive לבאקט Google Cloud Storage שלך

הגדרות OAuth Google

Client ID מ-Google Cloud Console > APIs & Services > Credentials
Client Secret מאותו OAuth 2.0 Client ID
מזהה הפרויקט ב-Google Cloud שבו נוצר ה-OAuth Client

הגדרות העברה

העתק את המזהה מכתובת ה-URL של התיקייה ב-Google Drive
שם הבאקט ב-Google Cloud Storage שאליו תרצה להעביר את הקבצים
מזהה הפרויקט ב-Google Cloud Platform שבו נמצא הבאקט

Service Account Key (JSON)

העתק את תוכן קובץ ה-JSON של Service Account מ-Google Cloud Console > IAM & Admin > Service Accounts

הוראות הגדרה

1. יצירת OAuth Client ID:

• עבור אל Google Cloud Console - Credentials

• לחץ "Create Credentials" > "OAuth client ID"

• בחר "Web application"

• הוסף Redirect URI: http://localhost:8080/oauth2callback


2. יצירת Service Account:

• עבור אל Service Accounts

• צור Service Account חדש

• הוסף תפקידים: Storage Admin, Drive API User

• צור מפתח JSON והעתק את התוכן


3. הפעלת APIs:

• Google Drive API

• Google Cloud Storage API
